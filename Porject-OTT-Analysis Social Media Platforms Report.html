OTT-Analysis Social Media Platforms Report

The project involves the analysis of Over-the-Top (OTT) platforms, focusing on their presence and performance on social media platforms. The report provides insights into how OTT platforms engage with audiences through social media, including key metrics and trends. The analysis involves data extraction, using web scraping techniques, and detailed reporting of findings related to OTT platforms' social media strategies. This project highlights the use of data analysis in understanding digital media platforms' performance in the social sphere.
